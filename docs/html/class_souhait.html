Detailed Description

Cette classe décrit le souhait.

Le souhait représente une recherche faite par un client acheteur. Cette classe représente le souhait et propose des méthodes permettant de le créer, de le modifier ou encore de le sauvegarder dans la base. Cette classe ne propose aucune interface graphique.
