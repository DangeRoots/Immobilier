Detailed Description

Cette classe décrit le client qu'il soit acheteur ou vendeur.

Le client est l'objet central du logiciel. Sans client, il est impossible de faire quelque action que ce soit. La classe Client permet de le représenter et de l'instancier, mais aussi de le sauvegarder dans la base.
