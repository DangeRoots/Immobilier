Detailed Description

Cette classe permet les échanges avec la base de données.

La base de données est accessible depuis les objets métier, tels que le client, le souhait ou le bien. Les instances de ces objets n'ont pas connaissance des paramètres de la base qui sont chargés par la classe BDD. Cette classe propose des méthodes permettant de créer une instance de base, de vérifier qu'une connexion est ouverte ou de fermer une connexion existante.
