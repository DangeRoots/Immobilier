Detailed Description

Classe d'instanciation des paramètres utilisateur.

La classe Parametre permet de charger le fichier des paramétres utilisateur. Elle s'appuie sur un fichier de paramétrage au format xml. Dans cette version, les paramètres ne sont ni affichables ni modifiables dans l'application.
