Detailed Description

Cette classe décrit la ville.

La ville est utilisée dans de nombreux objets de l'application, tels que le client, le bien ou le souhait. Cette classe représente la ville et propose des méthodes permettant d'y accéder. Cette classe ne propose aucune interface graphique, ni de méthodes permettant de créer de nouvelles villes ou de modifier les villes existantes.
